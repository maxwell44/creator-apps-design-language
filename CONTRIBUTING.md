# Contributing

Creator Apps Design Language 当前处于早期阶段。

提交建议时，请说明：

1. 需要解决的真实产品问题。
2. 该规则适用于哪些 Creator Apps。
3. 是否会增加用户认知成本。
4. 是否包含无障碍与边缘状态。
5. 是否与现有原则冲突。

我们优先接受：

- 来自真实产品的验证；
- 可复用的设计规则；
- 明确的 Do / Don't；
- 无障碍补充；
- 开发可执行规范。

我们不优先接受：

- 只基于个人审美的颜色修改；
- 单张概念图；
- 没有真实场景的动画建议；
- 为了追逐系统风格而添加的视觉效果。
