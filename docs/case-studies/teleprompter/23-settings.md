# 23 设置页规范

## 定位

主设置页承载订阅状态、全局默认值、语言、帮助与反馈、法律和关于。它不承担当前 Session 的临时控制。

## 建议分组

### Pro
当前状态、升级或管理订阅、恢复购买。顶部区域简洁，不做大型营销 Banner。

### Teleprompter Defaults
默认字号、默认速度、默认倒计时、默认镜像或对齐。可分阶段补充，本轮 UI 可先保留现有能力。

### General
Language、Appearance 与其它通用偏好。

### Support
Help、Feedback、Rate App、What’s New。

### About
Privacy、Terms、Version、Other Apps（低权重）。

## 视觉

接近原生 iOS Settings 的信息结构：少卡片、少彩色图标、清晰组标题、稳定行高、统一 Disclosure、Toggle 与 Value。

## Pro

Pro 功能表达价值，不使用强促销语言。已订阅用户优先看到状态与管理，而不是升级广告。

## 不做

- 不把每个 Section 包成大卡；
- 不使用多彩渐变；
- 不让“其他作品”比核心设置更显眼；
- 不隐藏帮助和订阅状态入口。
