# 15 导航与工具栏

## 导航语义

- **Push**：用于信息层级深入，返回后保持上层上下文。
- **Sheet**：用于临时任务、选择与上下文设置。
- **Full Screen**：用于沉浸式创作 Session，如提词、录制、预览。
- **Overlay**：用于短暂控制和状态，不承担完整页面导航。

## Back、Close、Done

- Back：返回上一级内容层级。
- Close：退出临时模态任务。
- Done：确认编辑或设置并关闭。
- Cancel：放弃未完成的临时任务。

不得混用。

## Floating Toolbar

提词与录制场景中的浮动工具栏应：

- 把相关控件放在同一容器；
- 不遮挡阅读眼线；
- 高频控件常驻；
- 低频控件进入 Sheet；
- 支持自动隐藏；
- 触摸屏幕后恢复；
- 状态变化不移动脚本。

## Tab Bar

Tab 代表长期稳定的顶层目的地，而不是功能按钮。如果两个 Tab 的业务边界难以理解，应优先明确命名与定位。

## 禁止事项

- 同类设置有时 Sheet、有时全屏 Overlay；
- 连续任务中多层全屏 Present；
- 每页重新发明返回按钮；
- 把低频功能全部放进底部工具栏。
