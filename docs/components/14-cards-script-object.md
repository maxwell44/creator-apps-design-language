# 14 卡片与脚本对象

## Script Object 不是普通卡片

脚本是内容对象，颜色是其视觉记忆与组织方式。设计需要支持快速识别、快速浏览、快速进入，并在脚本数量增加后仍然有效。

## Anatomy

- Color Marker / Colored Surface
- Title
- Content Preview
- Metadata：最近编辑、字数或预计时长
- Optional State：收藏、最近使用、同步状态
- More Menu

## Card 与 Row

Card 适合视觉浏览、双列或自适应网格；Row 适合搜索结果、大量脚本和紧凑列表。两者可以共存，但信息顺序与语义必须一致。

## 颜色策略

不取消多颜色能力。使用低饱和色面、清晰 Accent 和高对比标题；颜色与编辑器共享同一 ID，并支持颜色名称、图标或标签。

## 交互

点击主体进入提词；更多菜单承载编辑、收藏、删除。不要把打开、提词、编辑全部放成常驻按钮。

## 禁止事项

- 全部改成白色同质卡片；
- 整卡高饱和渐变；
- 用阴影代替边界；
- 堆积多个小按钮；
- 颜色没有一致语义。
