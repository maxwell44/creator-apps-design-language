# 13 按钮

## Variants

### Primary
每个页面唯一或最重要的动作。使用品牌色或高对比填充，高度建议 48–52 pt，并支持加载与禁用状态。

### Secondary
用于次级但重要动作。采用 Tinted 或中性 Surface，不与 Primary 竞争。

### Tertiary / Borderless
用于导航、取消和轻量操作，无独立容器或使用极弱背景。

### Destructive
只用于删除、丢弃等不可逆动作，使用系统危险色和明确文案。

### Glass Control
只用于媒体与沉浸场景中的浮动控制，触控区域至少 44×44 pt，相关操作组合在同一容器。

## 内容规则

按钮使用明确动词，如“创建脚本”“开始录制”“保存视频”。避免模糊的“确定”“继续”。

## Loading

加载时保持按钮宽度、防止重复提交，并显示 Activity 或进度。

## 禁止事项

- 同屏多个相同权重的彩色按钮；
- 把所有功能都设计成圆形图标；
- 按钮高度不足 44 pt；
- 用渐变作为默认 Primary。
