# 16 设置组件

## 两类设置

### Global Defaults
长期默认偏好，如默认字号、滚速、倒计时、视频质量、语言和外观。

### Session Controls
当前一次提词或录制中的临时调整，如当前速度、字号、镜像和提词位置。

两者必须在信息架构和文案上区分。

## Setting Components

- Setting Group
- Setting Row
- Toggle Row
- Slider Row
- Picker Row
- Value Disclosure Row
- Pro Locked Row
- Destructive Row
- Help Text

## 视觉

优先使用系统分组结构，不给每行单独大卡片，不使用多彩装饰图标。组标题简短，说明文字放在组下方。

## Pro 功能

锁定状态应说明价值，不只显示皇冠。点击后先展示上下文说明，再决定是否进入完整付费墙。

## 禁止事项

- 主设置页成为营销页面；
- 设置名称使用技术术语；
- 多套不同 Settings Row；
- 每个开关都有独立卡片和阴影。
