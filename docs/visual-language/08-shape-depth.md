# 08 圆角、描边与层级

## Shape Tokens

| Token | Value | 用途 |
|---|---:|---|
| radius-s | 8 | 小标签、紧凑选项 |
| radius-m | 12 | 行容器、小按钮 |
| radius-l | 16 | 普通卡片、分组 |
| radius-xl | 20 | 大型内容卡、浮动面板 |
| radius-2xl | 24 | 关键 Glass Toolbar |
| radius-full | 999 | 圆形与胶囊 |

## 层级表达顺序

优先使用空间与留白，其次是背景色差、描边、材质，最后才是阴影。

## 描边与阴影

普通描边保持 0.5–1 pt。阴影只用于真正浮在内容上方的临时工具栏、拖拽对象或关键浮动操作。脚本卡、设置行和普通内容容器默认无阴影。

## 卡片

卡片只用于表达独立内容对象、可操作信息组或真实空间层级。不要把页面所有内容都包进卡片。

## 禁止事项

- 渐变、阴影、描边和 Glass 同时叠加；
- 每个页面发明新的圆角；
- 普通设置行使用大圆角独立卡片；
- 通过厚重阴影制造高级感。
