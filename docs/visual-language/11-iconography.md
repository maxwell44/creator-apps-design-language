# 11 图标系统

## 默认选择

优先使用 SF Symbols。只有品牌标志、系统图标无法准确表达的核心概念，或需要建立独特产品识别时使用自定义图标。

## 风格

- 同一工具栏使用一致 Weight；
- 默认 Regular 或 Medium；
- 避免混用线性、填充与多彩图标；
- 选中态可从 Outline 转为 Fill；
- 图标大小由语义决定。

## 常用尺寸

- 紧凑辅助：14–16 pt；
- 设置行：18–20 pt；
- 工具栏：20–22 pt；
- 主要圆形控制：22–26 pt；
- 空状态：36–48 pt。

## 图标与文字

只显示图标时必须有 Accessibility Label，且语义足够常见。不常见功能优先使用“图标 + 文字”。

## 禁止事项

- 每个设置图标不同颜色；
- 使用 Emoji 代替系统图标；
- 混用不同粗细；
- 添加无操作意义的装饰图标。
