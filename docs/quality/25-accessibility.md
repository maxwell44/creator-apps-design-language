# 25 无障碍

## Dynamic Type

普通 UI 全量支持 Dynamic Type。提词文字使用独立可调系统，但标签、按钮、设置和说明仍需响应 Dynamic Type。

## VoiceOver

所有图标按钮提供稳定 Label。录制开始、倒计时、录制停止、保存成功、导出失败与语音跟随状态需要可被辅助功能理解。

## Differentiate Without Color

彩色脚本增加第二通道：标签、图标、边线模式或颜色名称。录制状态不能只依赖红色。

## Reduce Motion

取消大幅位移、视差、连续漂浮和装饰性入场，保留必要淡入淡出与状态变化。

## Reduce Transparency

Glass 自动转为更不透明 Surface，保证文字、图标和工具栏边界清晰。

## Increase Contrast

增强 Secondary Text、Divider、Glass Stroke、选中状态和脚本卡边界。

## Touch 与 Voice Control

所有触控目标至少 44×44 pt。高频按钮提供简短稳定的输入名称，如播放、暂停、录制、停止、调速和返回。

## 测试矩阵

每个 P0 页面至少测试：Light/Dark、Small/Default/Accessibility XXL、Reduce Motion、Reduce Transparency、Increase Contrast、Differentiate Without Color、VoiceOver、竖屏与横屏。
