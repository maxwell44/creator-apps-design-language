# Creator Apps Design Language

> **Less Interface. More Creation.**  
> 少一点界面，多一点创作。

Creator Apps Design Language（CADL）是一套面向创作者工具的产品与视觉设计语言。

它不追求“看起来像一张漂亮的概念图”，而追求真实产品中的克制、统一、可用性与完成度。它首先服务于灵感提词器，也可以扩展到图像、视频、摄影、写作等 Creator Apps。

## 核心信念

优秀的创作工具不会争夺注意力。

它们让创作者更快开始、更少犹豫、更顺利完成作品。当工具发挥作用时，用户应该专注于内容、表达和结果，而不是软件本身。

> 我们设计的不是界面，而是创作者完成作品时的信心。

## 设计判断顺序

当两个方案都“看起来不错”时，按以下顺序选择：

1. 更容易理解的方案。
2. 更少步骤的方案。
3. 更舒适的方案。
4. 更统一的方案。
5. 最后才是更有视觉表现力的方案。

## 非目标

CADL 不试图：

- 复制 Apple、谜底科技或任何具体产品的视觉外观。
- 把所有 Creator App 设计成完全相同。
- 用设计系统替代真实用户研究。
- 为了统一而牺牲特定工具的核心能力。
- 用 Liquid Glass、渐变或动画证明“设计感”。
